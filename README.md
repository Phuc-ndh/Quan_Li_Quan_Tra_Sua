# Quản lý quán trà sữa
## Description
- Mục tiêu: Hỗ trợ chủ quán, nhân viên quản lí các hoạt động của quán trà sữa.
- Tính năng: 
  * Xem được trạng thái đơn hàng hiện tại
  * Thêm món, xóa món
  * Tính tiền với hóa đơn có các thông tin:
    1. Tên món
    2. Số lượng
    3. Giá 
    4. Thông tin quán
  * Thống kê thông tin bán hàng
